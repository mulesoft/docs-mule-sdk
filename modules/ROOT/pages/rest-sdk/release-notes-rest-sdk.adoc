REST SDK is a tool that allows you to create connectors based on an API specification supplying additional connector capabilities via a connector descriptor.

== 0.5.0

*July 19, 2021*

=== What’s New

* Added code regeneration capabilities for REST SDK projects. (RSDK-484)
* It is now possible to generate custom operations as Composer sidecars. (RSDK-482)

=== Fixed Issues

[%header%autowidth.spread]
|===
|Issue|Number
|Fixed a NullPointerException that occurred when working with REST SDK 0.5.0. | (RSDK-495)
|Generated class names for ENUMS that no longer collide with each other. | (RSDK-514)
|Fixed the RestSampleDataProvider definition. | (RSDK-499)
|===

== 0.2.1

*May 21, 2021*

=== What’s New

* Min Mule version is now 4.3.0 by default for the generated connectors (RSDK-393)
* Added a new validation engine for simple composition (RSDK-401)
* Operation display names declared in the API specs are now used to generate the operation display name (RSDK-412)
* Added support for PREMIUM and CERTIFIED connector categories (RSDK-90)
* Added a license.md file generation for corresponding categories (RSDK-216)

=== Fixed Issues

[%header%autowidth.spread]
|===
|Issue|Number
|Evaluation licenses are now correctly generated |(RSDK-218)
|Trigger parameters marked as optional are now generated optionally | (RSDK-410)
|Renamed TLS tab in the connector configuration to Security tab |(RSDK-238)
|===
